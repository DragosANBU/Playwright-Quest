# Playwright Automation Exercise Project

## This project was made as a requirement exercise for an internship. 
### - The first requirement is found as a .md file named SIMPLE_TEST_PLAN.md in the project 
### - The second requirement is using Playwright, C# and NUnit.
### The goal of this project is to automate the most important scenarios of https://automationexercise.com website, such as: 
- registration and login 
- browsing products 
- adding products to cart 
- checkout 
### You can run the test by following the next steps: 
1. Clone repository 
2. Open the project using your preferred IDE (e.g. Visual Studio) or by navigating to the folder:
```bash
cd Playwright-Quest
```
3. Restore dependencies:
```bash
dotnet restore
```
4. Build the project:
```bash
dotnet build
```
5. Install playwright browsers (You can change .net version with your prefereed one):
```bash
pwsh bin/Debug/net10.0/playwright.ps1 install
```
6. Run tests:
```bash
dotnet test
```
**Or you can complete the steps using your IDE interface.**
